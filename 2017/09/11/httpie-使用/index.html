使用 httpie 测试http请求

2017年9月11日

httpie 的使用

安装

brew install httpie

测试 GET请求

http GET target-url

测试 POST json 请求

http POST target-url field1=val1 field2:=val2

field2:=val2： 表示field2是一个整型数值

测试 PUT 请求

http PUT target-url field=val

测试DELETE 请求

http DELETE target-url
