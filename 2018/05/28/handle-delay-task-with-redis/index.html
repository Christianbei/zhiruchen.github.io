思路

生产者发送一个定时任务到sorted set队列，score为未来的执行时间；这样队列第一个数据就是最近要执行的(消费)的任务

消费者定时(一个很短的时间间隔)去队列取出第一个任务，看是否到执行时间(score), 如果到了就执行任务，并将这个任务从队列删掉。
