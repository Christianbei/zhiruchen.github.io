golang unit test

2018年4月17日

basics

  • 测试的代码文件和源代码文件在同一目录下, 名称为xxx_test.go，比如测试fib.go文件，则测试文件为fib_test.go

  • 测试函数名称为func TestXXX(t *testing.T)

  • 使用测试用例表是测试代码更整洁

性能测试

  • 在测试代码中以func BenchmarkYourFunc(b *testing.B)开头

example

现在有一个测试计算第n个fib数的函数

fib_test.go

go test 测试结果

go test -v

测试覆盖率

性能测试

指定测试匹配名称的函数

go test -run=FuncName
